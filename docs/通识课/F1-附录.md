### 00-课程介绍

- 删除：`Delete`
- 复制: `Ctrl + C(opy)`
- 粘贴: `Ctrl + V`
- 重命名: `F2`
- 切换中英文输入法: `Shift`
- 切换输入法: `Win + Space`
- 保存: `Ctrl + S`
- 另存为: `Ctrl + Shift + S`
- 新建文件夹: `Ctrl + Shift + N`
- 打开文件资源管理器: `Win + E(xplore)`
- 新建桌面：`Win + Tab`
- 切换桌面：`Win + Ctrl + ⬅️/➡️`

- Edge 浏览器

  - `Ctrl + D`：将[教案](https://share.fzf404.top)加入收藏夹
  - `Ctrl + H`：历史记录
  - `Ctrl + J`：下载列表

- vscode 插件
