## 说明

1. 听完一节课后根据教案做一遍，不要边听边做。
2. 课程中所有用到的快捷键都会在 `F1-快捷键` 中列出。
3. ++win+tab++ 新建桌面，++win+ctrl+left++ / ++right++ 切换桌面。
4. 希望大家做一个乐于分享自己知识的人。

## 特色

1. 网络上没有一门前后端运维一起讲的通识课。
2. 仅用 10 节课，完成一个完整的项目。
3. 前端(jQuery) + 后端(Python) + 运维(Ubuntu)
