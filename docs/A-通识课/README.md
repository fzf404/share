## 说明

1. 听完一节课后根据教案做一遍，不要边听边做。
2. 课程中所有用到的快捷键都会在`F1-附录`中列出。
3. `Win+Tab`新建桌面，`Win+Ctrl+⬅️/➡️`切换桌面，`F11`浏览器全屏
4. 希望大家做一个乐于分享自己知识的人。
