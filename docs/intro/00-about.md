## 课程说明

- 听完一节课后根据教案做一遍，不要边听边做。
- 课程中所有用到的快捷键都会在 `F1 快捷键` 中列出。
- ++win+tab++ 新建桌面，++win+ctrl+left++ / ++right++ 切换桌面。
- 希望大家做一个乐于分享自己知识的人。

## 课程特色

- 前后端运维一起讲的通识课。
- 仅用十节课完成一个完整的项目。
- 前端 ( jQuery ) + 后端 ( Python ) + 运维 ( Ubuntu )
